# Basket
Scrip para **automatizar** procesos de youtube, noche de programación y cración de contenidos de ALSW

## Heramienta CLI

```bash
 basket-cli -h
usage: basket-cli [-h] [--icono ICONO] [--blender_proxy] [--blender_renderizar] [--blender_borrar]
                  [--blender_completo] [--proyectovideo PROYECTOVIDEO]
                  [--proyectoarticulo PROYECTOARTICULO] [--file FILE]

Heramientas Automatizacion de ALSW

optional arguments:
  -h, --help            show this help message and exit
  --icono ICONO, -i ICONO
                        Actualizar Icono a folder gtk
  --blender_proxy, -bp  Creando proxy de Blender
  --blender_renderizar, -br
                        Renderizar video con Blender
  --blender_borrar, -bb
                        Borrar Temporales de Blender
  --blender_completo, -bc
                        Renderiza video Blender y sube a youtube
  --proyectovideo PROYECTOVIDEO, -p PROYECTOVIDEO
                        Crear Folder proyecto de Video
  --proyectoarticulo PROYECTOARTICULO, -a PROYECTOARTICULO
                        Crear articulo base
  --file FILE, -f FILE  Archivo trabajar

```
